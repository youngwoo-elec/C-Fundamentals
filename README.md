# 🚀 C Fundamentals
> **"A comprehensive study of C programming and core logic"**

본 저장소는 군 복무 기간 중 **'A Book on C'**를 기반으로 C언어의 전반적인 내용을 복기하고, 
전자공학(EE) 전공자에게 필수적인 메모리 구조 및 포인터 제어 능력을 완벽히 내재화하는 공간입니다.

---

## 📌 Key Objectives
* **Memory Logic:** Stack, Heap, Data 영역의 메모리 할당 원리 심층 이해
* **Pointer Mastery:** 포인터 연산과 배열의 관계를 이용한 저수준 데이터 처리 숙달
* **Data Structures:** 구조체(Struct)와 공용체(Union)를 활용한 하드웨어 제어 논리 학습
* **Daily Consistency:** 제한적인 환경에서도 꾸준한 학습과 커밋(Commit) 기록 유지

## 📂 Directory Structure
* `01-Basic-Syntax`: 연산자, 제어문 등 기초 문법 복기
* `02-Functions`: 함수 설계 및 재귀 알고리즘 분석
* `03-Pointers`: (Core) 포인터 심화 및 메모리 주소 체계 연구
* `04-Structs`: 하드웨어 레지스터 제어 기반 데이터 구조 설계
* `Review-Log`: 학습 후 기술적 고찰 및 회고 기록

## 🛠 Tech Stack & Environment
* **Language:** C
* **IDE:** GitHub Codespaces
* **Compiler:** GCC
